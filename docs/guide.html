Build Your Own Chart

This guide will help you create your own SVG chart using the TypeScript chart library.

1. Set Up Your HTML

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <title>My Chart</title>
</head>
<body>
  <div id="chart"></div>
  <script type="module" src="my-chart.ts"></script>
</body>
</html>

2. Create Your Chart Script

In my-chart.ts (or any .ts file), import the chart you want to use and provide your data:

import { PieChart } from '../src/charts/PieChart';
import { LabelValueDataPoint } from '../src/models/ChartTypes';

const data: LabelValueDataPoint[] = [
  { label: 'A', value: 10 },
  { label: 'B', value: 20 },
  { label: 'C', value: 30 },
];

const options = {
  width: 600,
  height: 400,
  title: 'My Custom Pie Chart',
  showLegend: true,
};

const chart = new PieChart(options, data);
const chartContainer = document.getElementById('chart');
if (chartContainer) {
  chartContainer.innerHTML = chart.render();
}

3. Customize Your Chart

  • Change options to adjust size, colors, title, etc.
  • Use different chart types (BarChart, LineChart, etc.)
  • Style your chart with CSS for a custom look

4. Download Your Chart

To let users download the SVG, add a button and use this function:

function saveSvgToFile(svg: string, filename: string) {
  const blob = new Blob([svg], { type: 'image/svg+xml' });
  const url = URL.createObjectURL(blob);
  const link = document.createElement('a');
  link.href = url;
  link.download = filename;
  document.body.appendChild(link);
  link.click();
  document.body.removeChild(link);
  URL.revokeObjectURL(url);
}
